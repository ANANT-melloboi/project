# EduCom - Performance Analysis Web App

EduCom is a Streamlit-based web application that allows students to analyze their academic performance by predicting their CGPA based on their internal and external marks across multiple terms/semesters.

## Features
- User Authentication: Create an account and login to access the performance analysis.
- Performance Prediction: Predict CGPA based on average internal and external marks for each term.
- Visualization: View predicted performance trends through a line plot.
- Database Support: Store user credentials securely using SQLite and SHA256 hashing.
- Customizable: Supports both school and college levels with dynamic term/semester selection.

## Technologies Used
- Python
- Streamlit
- SQLite
- Scikit-learn
- Matplotlib
